# TerraSync Grounds Analysis

Public, data-driven benchmark of autonomous grounds-maintenance equipment — the value analysis
Wesley sends on sales calls, hosted for easy sharing.

**V0:** robotic mower lineup value analysis (26 models, Kress / Husqvarna / NEXMOW).
Data sourced from Velocity production: `mowerCatalog.ts` + optimizer `mower_specs.py`.

Served at GitHub Pages. Source of the concept + roadmap:
`terrasync-web/docs/research/grounds-analysis-benchmark-page-scoping-2026-07-01.md`.

`index.html` is a self-contained single file (embedded images, CDN Chart.js). `noindex` is set so
the page is shareable by link but not search-indexed.
